# FinanceManager
Приложение на Android для организации личного бюджета на языке Java с использование SQLite

Приложение упрощает организацию личного бюджета, помогает в обработке информации о финансах: ускоряет работу по составлению личных отчетов, позволяет вести учет доходов и расходов, планирование бюджетов на различные категории, анализировать финансовые данные. Все операции выполняются локально на устройстве без необходимости подключения к интернету. Множество аналогичных сервисов имеют облачные подключения, требуют подписки, и зависят от сети.
# Руководство по эксплуатации

Процесс эксплуатации приложения прост и интуитивно понятный, чтобы каждый пользователь мог легко адаптироваться к его функционалу и получать от него максимальную пользу. Для начала использования приложения необходимо войти в аккаунт или зарегистрироваться, на рисунке находится экран авторизации для ввода логина и пароля, далее при нажатии на кнопку «Вход» происходит вход в аккаунт, если аккаунта нет, его необходимо зарегистрировать, нажав на кнопку «Регистрация» после чего ввести новые данные для авторизации.

Перейдя на следующую активность, нажав кнопку «Доход» или «Расход» открывается экран для добавления транзакции, первая возможность, это выбор даты (автоматически выбирается сегодня), выполнен в виде календаря для удобной и быстрой записи. Всплывающие списки кошельков, а также категорий расходов или расходов работает по одинаковому принципу, при нажатии открывается список, который позволяет выбрать необходимый кошелек или категорию при нажатии на нее список закрывается и запоминает выбор. Далее необходимо ввести сумму операции и по желанию ввести заметки в строку «Описание».

Для сохранения записи необходимо нажать на кнопку «Сохранить». При добавлении описания транзакции в столбце «Доп» записывается «…».

Для редактирования, добавления и удаления данных необходимо нажать на нужную строку в таблице на главном экране, после ввода новой информации, необходимо нажать на кнопку «Изменить данные». Либо для удаления записи нажать кнопку «Удалить». После нажатия на кнопку «Изменить данные» или «Удалить» необходимо подтвердить свои действия. Для отмены действия необходимо нажать «Нет» или для подтверждения действия «Да», после чего изменения сохраняться в базу данных, откроется главная активность.

Для добавления нового кошелька/категории необходимо нажать кнопку рядом со списком кошельков/категории при добавлении или редактировании. После нажатия на кнопку для добавления появиться окно:
- «Добавить новый кошелек», куда необходимо ввести название кошелька и валюту создаваемого кошелька;
- «Новая категория», в нем необходимо записать название новой категории и выбрать к какому типу будет относится категория «Доход» или «Расход».

После нажатия кнопки «Сохранить» кошелек/категория будут сохранены и их можно будет использовать для записи новых транзакций.

Для просмотра отчетности о доходах и расходах необходимо на главном экране нажать на знак «Меню» (слева сверху три горизонтальные линии "Гамбургер") и перейти на вкладку «Отчеты». На экране видна диаграмма доходов кошелька с названием «Кошелек». Для выбора даты необходимо воспользоваться календарем, при изменении кошелька в верхней части экрана изменяются и выводимые данные.

Для просмотра «Бюджета» необходимо выбрать советующий элемент, выбрав его в меню. На экране, можно выбрать необходимый кошелек, добавить или редактировать бюджет. Для добавления «Бюджета» необходимо нажать на кнопку «+» (справа снизу экрана), после ввода данных (как при добавлении/редактирования транзакций), нажать на кнопку «Сохранить».
